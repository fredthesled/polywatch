🔌 API Reference
GET
https://gamma-api.polymarket.com/markets?closed=false&limit=100&offset=N
Active markets. Returns id, question, slug, conditionId, clobTokenIds[], endDate, liquidity, volume, outcomePrices[], active. Rate limit: ~60 req/min.
GET
https://data-api.polymarket.com/trades?market={conditionId}&limit=50
Trades per market. Returns proxyWallet, side (BUY/SELL), size, price, outcome, timestamp (unix), transactionHash. Rate limit: ~30 req/min.
CORS
https://corsproxy.io/?url={encoded_url}
Used as CORS proxy for browser-side API calls. Falls back through three proxy services; shows empty trade list if all fail. Session cache via sessionStorage keyed by conditionId + wallet.
